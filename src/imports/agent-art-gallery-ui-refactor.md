Refine the current **Agent Art Gallery** spatial UI to keep the same overall framework (visible world + zones + agents + hotspots), but significantly improve the **art direction and scene design quality**.

## Main goal

Keep the current spatial interaction framework, but make the design feel:

* **less “AI-generated purple dashboard”**
* more like a **designed pixel-art world / gallery space**
* more cohesive, grounded, and intentionally art-directed

## What to preserve (important)

* The visible spatial world layout and zone concept
* Agent avatars and in-world interactions
* Hotspots for Gallery / Commissions / Studio / Exchange / Lounge
* The idea of zone-focused screens
* Overlay UI panels as secondary elements

## What must change

### 1) Color palette (critical)

Replace the current heavy neon-purple scheme with a more grounded, designed palette.

Use a **museum + creative studio palette**:

* Deep charcoal / ink background (not pure black)
* Warm stone / muted cream highlights
* Desaturated plum ONLY as a secondary accent (not dominant)
* Dusty teal / slate blue for studio accents
* Moss/olive or cork accents for commission board
* Brass / amber for exchange and highlighted economy items
* Soft warm spotlight tones for gallery displays

### Palette rules

* Reduce overall saturation
* Avoid full-screen purple tinting
* Use accent colors by zone, not everywhere
* Prefer material-like colors over “cyber neon” colors

### 2) Scene design quality (critical)

Make each zone feel like a **real designed place**, not floating widgets.

Add environmental structure:

* floors / tiles / rugs / pathways
* walls / partitions / railings / counters
* display frames mounted on walls or stands
* bulletin board / cork board / terminal desk for commissions
* worktables / easels / monitors / drafting desks in studio
* market counter / display board / ticker board in exchange
* lounge seating / kiosk / leaderboard sign in agent lounge

### 3) Lighting and depth

Use lighting to improve the scene:

* gallery spotlights over featured works
* task lighting in studio area
* warmer accent lighting in exchange area
* subtle shadows under objects/avatars
* avoid generic glow halos around random labels

### 4) UI cleanup (less AI-looking)

Reduce “AI dashboard” visual patterns:

* fewer oversized glowing outlines
* fewer floating pills everywhere
* fewer generic metric cards on spatial screens
* fewer random gradients

Use cleaner, quieter UI treatment:

* thin pixel borders
* restrained highlights
* consistent panel hierarchy
* labels anchored to actual objects

### 5) Artwork presentation quality

Improve the gallery scene specifically:

* artworks should be displayed as curated framed pieces in a room
* create visual rhythm and spacing like a real exhibition
* featured artwork should have stronger composition and spotlighting
* include benches / signage / labels / viewing path to make it feel like a gallery
* avoid “grid of cards pasted into a dark box” feeling

### 6) Agent embodiment and presence

Keep agents visible, but make them feel integrated into the scene:

* place agents near meaningful objects (critic near review wall, artist near studio desk, collector near gallery/exchange)
* reduce label clutter
* use compact nameplates and hover states
* show presence/activity through subtle icons or animation hints, not large banners

---

## Composition guidance (to improve aesthetics)

* The world scene should feel like a designed environment first, UI second
* Use a clear focal point on each screen (e.g., featured gallery wall, studio workstation cluster, commission board, exchange counter)
* Build stronger foreground/midground/background layers
* Use empty space intentionally (not blank dark space)
* Keep overlays minimal until interaction

---

## Pixel style quality guidelines

* Consistent sprite scale and perspective
* Consistent tile grid
* No photorealistic or random stock-image elements in the world
* Pixel details should support mood and readability, not visual noise
* Pixel font only for headings/signage; readable UI font for body text/details

---

## Zone-specific style direction

### Gallery Hall

* Warm curated exhibition vibe
* Framed works, spotlight pools, wall labels, benches
* Feels premium and calm

### Commission Board

* Maker / workshop / bulletin vibe
* Cork board / pinned tickets / desk terminal
* Feels active and practical

### Studio Space

* Creative production room
* Worktables, screens, tools, draft wall, critique board
* Feels collaborative and busy

### Exchange Counter

* Market desk / licensing kiosk
* Wallet terminal, sales board, trend ticker
* Feels transactional but still artistic (not fintech dashboard)

### Agent Lounge

* Social waiting / discovery zone
* Seating, directory kiosk, role zones
* Feels comfortable and community-driven

---

## Final refinement target

The result should feel like:

* a **designed pixel-art creative world**
* with **professional art direction**
* and a **clear product UX framework**
  NOT like a dashboard with a pixel font theme.
